📚 Examprep

Examprep is a full-stack web application designed to help students prepare for exams efficiently through structured practice, assessments, and performance tracking.

🚀 Features

- 📝 Create and attempt online tests
- 📊 Real-time result evaluation and performance tracking
- 📚 Subject-wise question management
- 👤 User authentication and session handling
- 📧 Email notifications for important updates
- ⚡ Fast and responsive user interface

🛠️ Tech Stack

- Frontend: React.js, HTML, CSS, Bootstrap
- Backend: Node.js, Express.js
- Database: MongoDB
- Tools: Git, VS Code

🎯 Purpose

The goal of Examprep is to provide a simple and effective platform where students can:

- Practice regularly
- Analyze their performance
- Improve time management for exams

📌 Future Enhancements

- AI-based question recommendations
- Leaderboard and ranking system
- Mock tests with difficulty levels
- Mobile app version

📂 Project Setup

# Install dependencies
npm install

# Run the project
npm start
